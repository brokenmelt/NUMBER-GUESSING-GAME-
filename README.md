# Number Guessing Game 🎮

A simple Python number guessing game.

The program generates a random number between 1 and 100, and the player tries to guess it.

## Features
- Random number generation
- User input handling
- High/Low hints
- Loop until correct answer

## Technologies Used
- Python
- random module
